# 工具规格：glob (路径检索)

## 1. 功能目标
提供文件路径的快速匹配和定位能力。通过 `glob` 模式，代理可以快速筛选出符合命名规则的文件集合（如 `src/**/*.go`），从而精确锁定后续处理的目标文件。

## 2. 核心功能
*   **多维度过滤**：支持排除特定目录（如 `vendor/`, `.git/`）。
*   **多模式支持**：兼容标准 Glob 语法，支持递归、扩展匹配。
*   **按修改时间排序**：(可选) 优先返回最近修改的文件，帮助代理快速定位正在进行的变更。

## 3. 参数定义 (JSON)
*   **`pattern`** (string, 必填): Glob 路径匹配模式（如 `src/**/*.go`）。
*   **`case_sensitive`** (boolean, 可选): 路径匹配是否区分大小写，默认 `false`。

## 4. 返回值格式
*   **array of string**: 返回匹配的文件绝对/相对路径列表。
